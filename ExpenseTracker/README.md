<h1>Expense Tracker (Python)</h1>
<h2>Project Description</h2>
<p>
        The <strong>Expense Tracker</strong> is a console-based Python application that helps users
        record, store, and analyze their daily expenses. It allows users to add expenses,
        view all records, calculate total spending, and view category-wise summaries.
</p>
<h2>Objectives</h2>
<ul>
<li>Track daily expenses efficiently</li>
<li>Store expense data permanently using file handling</li>
<li>Analyze spending habits</li>
<li>Practice core Python programming concepts</li></ul>
<h2>Technologies Used</h2>
<li><strong>Programming Language:</strong> Python</li>
<h2>Features</h2>
<ul>
<li>Add new expenses with amount and category</li>
<li>View all recorded expenses</li>
<li>Calculate total expenses</li>
<li>Display category-wise expense summary</li>
<li>Menu-driven and user-friendly interface</li>
</ul>
<h2>Concepts Used</h2>
<ul>
<li>Functions</li>
<li>Loops and Conditional Statements</li>
<li>File Handling (Read and Write)</li>
<li>Dictionaries</li>
<li>Date Handling using <code>datetime</code> module</li></ul>
